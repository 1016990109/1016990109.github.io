Python 处理excel文件

  • python

网上搜索了下主要有几种方案：

1. 使用xlrd、xlwt、xlutils组合

这种方案比较常见，而且读取和写入速度较快，但是只能操作2003版本之前的xls文件，处理不了xlsx，所以想要处理2003版本之后的请绕道。
